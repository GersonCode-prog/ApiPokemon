# Pokémon Search

Este es un proyecto de búsqueda de Pokémon que utiliza la [PokeAPI](https://pokeapi.co/) para obtener información sobre los Pokémon.

## Funcionalidades

- **Búsqueda de Pokémon:** Permite buscar Pokémon por nombre.
- **Visualización de detalles:** Al hacer clic en un Pokémon, se muestra una ventana modal con detalles como ID, altura, peso, tipos, hábitat y color.
- **Filtrado por hábitat:** Los Pokémon se muestran con un color de fondo correspondiente a su hábitat.

## Tecnologías Utilizadas

- HTML
- CSS
- JavaScript

## Uso

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web.
3. Ingresa el nombre de un Pokémon en el campo de búsqueda.
4. Haz clic en un Pokémon para ver más detalles.
